# Intro to Postman with Zoom APIs

[Postman](https://www.getpostman.com) is a complete toolchain for API development and testing. As a third-party tool, it makes it easy for developers to test APIs in specially generated environments with imported testing collections.

For Zoom Marketplace developers, Postman provides a sample environment for testing Zoom APIs and services and authentication credentials with live Zoom data. The Zoom Marketplace provides a number of Postman collections which model the full features of the Zoom API, including a custom collection for Chatbots.

You can use [Postman](https://www.getpostman.com) for Web or download the [Desktop client](https://www.getpostman.com/downloads/).

## Zoom API

The **Zoom API Postman Collection** provides sample requests matching the design and specification of the Zoom API. Use this collection to explore available resources in the Zoom API and to test responses with live data.

Go to the [Zoom Developer Postman Workspace](https://www.postman.com/zoom-developer) to see all of the public Zoom Postman collections and choose one to get started.

## Zoom Chatbots

The **Zoom Chatbot Postman Collection** provides sample requests matching the design and specification of the Zoom Chatbot API. Use this collection to explore available resources in the Zoom Chat API and receive test responses with expected Chatbot message formats.

See [Get Started Fast with Zoom APIs](https://www.postman.com/zoom-developer/zoom-public-workspace/collection/rzauu5q/get-started-fast-with-zoom-apis) for a full walkthrough.
