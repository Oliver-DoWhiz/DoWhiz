# Chatbot API

-   **Version:** 2.0.0
-   **Host:** api.zoom.us/v2
-   **Protocols:** `https`
-   **Accepts:** `application/json`, `multipart/form-data`
-   **Responds With:** `application/json`, `application/xml`

More Info

-   **[Contact Us by Email](mailto:developersupport@zoom.us)**
-   **Contact URL:** &lt;https://developer.zoom.us/>
-   **Terms Of Service**:
    &lt;https://zoom.us/docs/en-us/zoom_api_license_and_tou.html>
-   **License:** [MIT for OAS 2.0](https://opensource.org/licenses/MIT)


The Zoom Chatbot API allows developers to access data related to Zoom chatbots.

You can use this API to build private or public chatbot applications on the
[Zoom App Marketplace](http://marketplace.zoom.us). To learn how to get
your credentials and create chatbot applications, read our
[Installation and Authentication Guide](/docs/team-chat/installation-and-authentication/).
All endpoints are available via `https` and are located at
`api.zoom.us/v2/`.
