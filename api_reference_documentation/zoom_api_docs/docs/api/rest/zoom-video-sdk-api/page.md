# Zoom Video SDK API

The Zoom Video SDK API allows owners of Zoom Video SDK accounts to access information from Zoom. You can use this API to build private services or applications. [Get your API key and secret](/docs/video-sdk/developer-accounts/#get-api-key-and-secret) to [make API requests](/docs/video-sdk/api-request/).

The API accepts `application/json` and `multipart/form-data`, and responds with `application/json` and `application/xml`. All endpoints are available at `https://api.zoom.us/v2/`.
