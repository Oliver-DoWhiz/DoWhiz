# Gmail API Samples

## Overview

This page documents sample code and projects available for Google Gmail API development through Google Workspace.

## Mail Importer for Gmail

### Description

Mail Importer for Gmail is an open source project from Google that uploads the contents of a Thunderbird mail archive to Gmail, doing its best to preserve the read state, flagged state, and folders of the messages.

### Technical Details

- **Language:** Java
- **Key API Endpoint:** Uses the Messages.import endpoint
- **Purpose:** Enables migration of email data from Thunderbird archives into Gmail while maintaining message metadata

### Access the Code

Developers can obtain the complete source code by:

1. Cloning the GitHub repository (https://github.com/google/mail-importer)
2. Browsing the Gmail-specific implementation (https://github.com/google/mail-importer/tree/master/src/main/java/to/lean/tools/gmail/importer/gmail)
