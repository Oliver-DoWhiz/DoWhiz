# Find Support for the Gmail API

## Overview

Google provides multiple support channels for Gmail API developers. Support resources are categorized by type of assistance needed.

**Important Note:** These channels should only be used for _developer_ issues. Feedback about the core product should be submitted via the **Feedback** or **Report a problem** links in the product user interface.

## Questions & Advice

### Community Forums (Official)

The Google Workspace Developers community forum (https://discuss.google.dev/c/google-workspace/20) serves as the official space for conversations about Google Workspace development.

### Reddit (Unofficial)

Community-managed subreddits offer peer support:
- r/GoogleAppsScript (https://www.reddit.com/r/GoogleAppsScript/)
- r/googleworkspacedevs (https://www.reddit.com/r/googleworkspacedevs/)

### Stack Overflow

Technical questions can be posted on Stack Overflow (https://stackoverflow.com/questions/tagged/gmail-api) using the `[gmail-api]` tag. You can sign in with a Google Account.

## Developer Product Feedback

Users should search the Issue Tracker (https://issuetracker.google.com/issues?q=status:open%20componentid:191602%2B%20type:\(bug%7Cfeature_request%7Ccustomer_issue\)) before submitting feedback. You can:
- Star existing issues to indicate support
- Submit new bug reports
- Submit feature requests

## Direct Support

Google Workspace administrators can contact support specialists by emailing Google Workspace developer support.

**Include when contacting support:**
- Problem description and expected behavior
- Reproduction steps with code samples
- Expected vs. actual output and error messages
- Development environment details
